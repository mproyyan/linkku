## About Linkku
----------

Linkku is RESTful Application created using laravel framework that can be used for :
- Save and manage your link
- Shorten your url
- Create **Archive** for manage and collect links
  
<br>

## Instalation
----------

```console
   git clone https://github.com/mproyyan/linkku.git
```
```console
   composer install
```
